G4 Example 1: Allowing the content to be paused and restarted from where it was paused

Link to a simple Online Newspaper with Scrolling Text.

Source:

  • WCAG G4
  • SCR33: Using script to scroll content, and providing a mechanism to pause it

Tested on:

  • Windows 7, Firefox ESR 49.0.1, and JAWS 17
    • Result: When the user selects the Pause link the scrolling text stops. Selecting the Scroll link restarts the scrolling text.
